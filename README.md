# ❤️ Heart Stroke Prediction App

An end-to-end **Machine Learning project** that predicts the risk of heart stroke using multiple classification algorithms and is deployed as an interactive web application using Streamlit.



## 🚀 Project Overview

This project focuses on building a predictive system that can identify the likelihood of a heart stroke based on patient health data.

The workflow covers the complete ML pipeline:
**EDA → Data Preprocessing → Model Training → Evaluation → Deployment**



## 🧠 Models Used

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Naive Bayes
* Decision Tree
* Support Vector Machine (SVM)



## 📊 Model Performance

| Model               | Accuracy | F1 Score |
| ------------------- | -------- | -------- |
| Logistic Regression | 87.5%    | 89.0%    |
| SVM                 | 86.41%   | 88.04%   |
| KNN                 | 85.33%   | 87.08%   |
| Naive Bayes         | 83.15%   | 84.58%   |
| Decision Tree       | 82.61%   | 83.84%   |

🏆 **Best Model:** Logistic Regression

* Highest Accuracy
* Best F1 Score (balanced precision & recall)

## 🔧 Installation & Setup

1. Clone the repository

```
git clone https://github.com/your-username/heart-stroke-prediction.git
cd heart-stroke-prediction
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the app


streamlit run app.py
